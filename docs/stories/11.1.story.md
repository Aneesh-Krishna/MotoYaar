# Story 11.1 — Admin Authentication

**Epic:** 11 — Admin Panel
**Story:** 11.1
**Status:** Draft
**Created:** 2026-03-15
**Depends On:** Story 1.7 (Vercel env vars configured), Story 1.2 (DB with Drizzle ORM)

## Story

**As an** administrator,
**I want** to log in to a separate admin area with credentials,
**so that** the admin panel is accessible only to authorised staff, completely separate from user accounts.

---

## Acceptance Criteria

1. - [ ] Admin accessible at `/admin` — all routes under `src/app/admin/**`
2. - [ ] Admin login form at `src/app/admin/login/page.tsx`: email + password fields
3. - [ ] Admin session managed by `iron-session` — separate from user NextAuth cookie
4. - [ ] `POST /api/admin/auth/login { email, password }` — verifies credentials, sets session cookie
5. - [ ] `POST /api/admin/auth/logout` — destroys admin session cookie
6. - [ ] Admin credentials stored in `admin_accounts` table (bcrypt-hashed password)
7. - [ ] Default admin seeded from `ADMIN_EMAIL` + `ADMIN_PASSWORD` env vars on first run via `src/scripts/seed.ts`
8. - [ ] Admin middleware (`src/app/admin/layout.tsx` or `src/middleware.ts` admin guard) protects all `/admin/**` routes; returns 404 (not 403) for non-admin requests
9. - [ ] Admin can create additional admin accounts from within the dashboard (Story 11.3 or separate action)
10. - [ ] `pnpm add iron-session bcryptjs && pnpm add -D @types/bcryptjs` — install before coding
11. - [ ] `pnpm typecheck` and `pnpm lint` pass

### Seed Script

[Source: architecture.md#10-background-jobs]

Create `src/scripts/seed.ts`:

```typescript
import { db } from "@/db";
import { adminAccounts } from "@/db/schema";
import bcrypt from "bcryptjs";
import { eq } from "drizzle-orm";

async function seed() {
  const adminEmail = process.env.ADMIN_EMAIL;
  const adminPassword = process.env.ADMIN_PASSWORD;

  if (!adminEmail || !adminPassword) {
    console.log("ADMIN_EMAIL or ADMIN_PASSWORD not set — skipping admin seed");
    return;
  }

  const existing = await db.query.adminAccounts.findFirst({
    where: eq(adminAccounts.email, adminEmail),
  });

  if (existing) {
    console.log("Admin account already exists — skipping");
    return;
  }

  const hash = await bcrypt.hash(adminPassword, 12);
  await db.insert(adminAccounts).values({ email: adminEmail, passwordHash: hash });
  console.log(`Admin account created: ${adminEmail}`);
}

seed().then(() => process.exit(0)).catch(err => {
  console.error(err);
  process.exit(1);
});
```

Add to `package.json` scripts:

```json
"scripts": {
  "seed": "tsx src/scripts/seed.ts"
}
```

### iron-session Configuration

[Source: architecture.md#2-tech-stack]

Create `src/lib/adminSession.ts`:

```typescript
import { getIronSession, IronSessionData } from "iron-session";
import { cookies } from "next/headers";

declare module "iron-session" {
  interface IronSessionData {
    admin?: { id: string; email: string };
  }
}

const sessionOptions = {
  password: process.env.ADMIN_SESSION_SECRET!, // must be >= 32 characters
  cookieName: "motoyaar_admin_session",
  cookieOptions: {
    secure: process.env.NODE_ENV === "production",
    httpOnly: true,
    maxAge: 60 * 60 * 8, // 8 hours
  },
};

export async function getAdminSession() {
  return getIronSession<IronSessionData>(cookies(), sessionOptions);
}
```

Add `ADMIN_SESSION_SECRET` (32+ char random string) to environment variables.

### adminService

[Source: architecture.md#6-components]

Create `src/services/adminService.ts`:

```typescript
import bcrypt from "bcryptjs";
import { db } from "@/db";
import { adminAccounts } from "@/db/schema";
import { eq } from "drizzle-orm";

export const adminService = {
  async login(email: string, password: string): Promise<{ id: string; email: string }> {
    const admin = await db.query.adminAccounts.findFirst({
      where: eq(adminAccounts.email, email.toLowerCase()),
    });

    if (!admin) throw new Error("Invalid credentials");

    const isValid = await bcrypt.compare(password, admin.passwordHash);
    if (!isValid) throw new Error("Invalid credentials");

    return { id: admin.id, email: admin.email };
  },
};
```

Use a generic "Invalid credentials" error for both email-not-found and wrong-password cases (avoid user enumeration).

### Login API Route

[Source: architecture.md#5-api-specification]

Create `src/app/api/admin/auth/login/route.ts`:

```typescript
import { NextResponse } from "next/server";
import { adminService } from "@/services/adminService";
import { getAdminSession } from "@/lib/adminSession";

export async function POST(req: Request) {
  try {
    const { email, password } = await req.json();
    const admin = await adminService.login(email, password);

    const session = await getAdminSession();
    session.admin = { id: admin.id, email: admin.email };
    await session.save();

    return NextResponse.json({ ok: true });
  } catch (err) {
    return NextResponse.json({ error: "Invalid credentials" }, { status: 401 });
  }
}
```

### Admin Layout Guard

[Source: architecture.md#12-unified-project-structure]

Create `src/app/admin/layout.tsx`:

```tsx
import { getAdminSession } from "@/lib/adminSession";
import { notFound } from "next/navigation";

export default async function AdminLayout({ children }: { children: React.ReactNode }) {
  const session = await getAdminSession();

  if (!session.admin) {
    // Return 404 to obscure admin URL existence from non-admins
    notFound();
  }

  return (
    <div className="min-h-screen bg-gray-100">
      <AdminSidebar />
      <main className="ml-64 p-8">{children}</main>
    </div>
  );
}
```

Exception: the login page at `/admin/login` must NOT be protected. Use a separate layout or check the path:

```tsx
// In admin/layout.tsx, skip auth for login path
import { headers } from "next/headers";

const headersList = headers();
const pathname = headersList.get("x-pathname") ?? ""; // set by middleware
if (!session.admin && !pathname.endsWith("/login")) {
  notFound();
}
```

Alternatively, keep `admin/login/layout.tsx` as a separate layout with no auth guard.

### Admin Login Page

[Source: front-end-spec.md#3-screen-by-screen-specifications]

Create `src/app/admin/login/page.tsx` as a simple client form:

```tsx
"use client";
export default function AdminLoginPage() {
  const [email, setEmail] = useState("");
  const [password, setPassword] = useState("");
  const [error, setError] = useState("");
  const router = useRouter();

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    const res = await fetch("/api/admin/auth/login", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ email, password }),
    });
    if (res.ok) {
      router.push("/admin");
    } else {
      setError("Invalid email or password");
    }
  };

  return (
    <div className="min-h-screen flex items-center justify-center bg-gray-100">
      <div className="bg-white rounded-xl p-8 w-full max-w-sm shadow-sm">
        <h1 className="text-xl font-bold mb-6">Admin Login</h1>
        {error && <p className="text-red-500 text-sm mb-4">{error}</p>}
        <form onSubmit={handleSubmit} className="space-y-4">
          <input type="email" value={email} onChange={e => setEmail(e.target.value)}
            placeholder="Email" required className="w-full border rounded-lg px-3 py-2 text-sm" />
          <input type="password" value={password} onChange={e => setPassword(e.target.value)}
            placeholder="Password" required className="w-full border rounded-lg px-3 py-2 text-sm" />
          <button type="submit" className="w-full bg-gray-900 text-white py-2 rounded-lg text-sm font-semibold">
            Sign in
          </button>
        </form>
      </div>
    </div>
  );
}
```
