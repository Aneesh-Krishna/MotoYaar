# Story 14.1 — Public Trip Route Sharing

**Epic:** 14 — Growth & Engagement
**Story:** 14.1
**Status:** Draft
**Created:** 2026-04-28
**Depends On:** Story 13.2 (GPS Route Recording — `trip_routes` record must exist), Story 9.1 (Community post creation)

---

## Story

**As a** rider who completed a live trip with a recorded GPS route,
**I want** to share my route to the community feed and as an image card,
**so that** I can show others where I rode and attract new users to MotoYaar.

---

## Acceptance Criteria

1. - [ ] On the trip detail page (post-live-trip), a **"Share Route"** button is visible when `has_live_route = true`
2. - [ ] Tapping "Share Route" presents two options: **"Post to Community"** and **"Share Image Card"**
3. - [ ] **Post to Community:** pre-fills the new post form with the route map thumbnail as an image, the trip title as the post title, and the route stats (distance, duration, avg speed) in the description. User can edit before submitting. Post is tagged `Travel` + `Routes` automatically.
4. - [ ] **Share Image Card:** generates a static image (1080×1080px) containing the route polyline on an OSM map background, trip title, distance, duration, and a *"Tracked on MotoYaar"* watermark. User is presented with the OS share sheet to share the image.
5. - [ ] The route thumbnail used in posts is the bounding-box static map already generated for the Trips list (Story 13.3 — map thumbnail). No additional tile fetch needed.
6. - [ ] The share image card is generated client-side using `html2canvas` or equivalent — no server-side image generation required at MVP.
7. - [ ] Community posts created via "Post to Community" are identical to regular posts — fully editable, can be deleted, appear in feed.
8. - [ ] If the trip has no recorded route (`has_live_route = false`), the "Share Route" button is not shown.
9. - [ ] `pnpm typecheck` and `pnpm lint` pass

## Dev Notes

### Share Image Card (Client-Side Generation)

Use `html2canvas` to screenshot a hidden off-screen `<div>` that renders:
- Leaflet map centred on route bounding box with the polyline drawn
- Trip title, distance (km), duration, avg speed overlaid
- MotoYaar logo watermark (bottom-right corner)

```typescript
import html2canvas from 'html2canvas';

async function generateShareCard(routeData: TripRoute): Promise<Blob> {
  const el = document.getElementById('share-card-canvas');
  const canvas = await html2canvas(el!, { useCORS: true, scale: 2 });
  return new Promise(resolve => canvas.toBlob(blob => resolve(blob!), 'image/png'));
}
```

The hidden div is rendered at 540×540 CSS px (scale 2 → 1080×1080 output). It is positioned off-screen (`position: absolute; left: -9999px`) and removed from DOM after capture.

### OS Share Sheet

```typescript
const blob = await generateShareCard(route);
const file = new File([blob], 'my-ride.png', { type: 'image/png' });
if (navigator.canShare?.({ files: [file] })) {
  await navigator.share({ files: [file], title: trip.title });
} else {
  // Fallback: trigger download
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url; a.download = 'my-ride.png'; a.click();
}
```

### New Tag: "Routes"
