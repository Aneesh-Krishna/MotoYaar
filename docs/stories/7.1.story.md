**Epic:** 07 — Reports & Analytics

## Story

**As a** vehicle owner,
**I want** to see a visual breakdown of everything I've spent on a specific vehicle,
**so that** I can understand my spending patterns and identify the most costly areas.

## Dev Notes

### Route & Page Structure

[Source: architecture.md#12-unified-project-structure]

Create `src/app/(app)/reports/vehicles/[id]/page.tsx` as a Server Component. It fetches the report data server-side and passes it to the client chart/table components. The `[id]` segment is the vehicle ID.

```tsx
// src/app/(app)/reports/vehicles/[id]/page.tsx (RSC)
import { getServerSession } from "next-auth";
import { authOptions } from "@/lib/auth";
import { reportService } from "@/services/reportService";
import { redirect } from "next/navigation";

interface PageProps {
  params: { id: string };
  searchParams: { filter?: string; from?: string; to?: string };
}

export default async function VehicleSpendReportPage({ params, searchParams }: PageProps) {
  const session = await getServerSession(authOptions);
  if (!session) redirect("/login");

  const report = await reportService.getVehicleReport(
    params.id,
    session.user.id,
    searchParams
  );

  return <VehicleReportView report={report} vehicleId={params.id} />;
}
```

`VehicleReportView` is a `"use client"` component that owns chart-type toggle state and tab state.

### API Route

[Source: architecture.md#5-api-specification]

Create `src/app/api/reports/vehicles/[id]/route.ts`:

```typescript
import { getServerSession } from "next-auth";
import { authOptions } from "@/lib/auth";
import { handleApiError } from "@/lib/errors";
import { reportService } from "@/services/reportService";
import { NextResponse } from "next/server";

export async function GET(req: Request, { params }: { params: { id: string } }) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    const { searchParams } = new URL(req.url);
    const filter = searchParams.get("filter") ?? "all";
    const from = searchParams.get("from") ?? undefined;
    const to = searchParams.get("to") ?? undefined;

    const report = await reportService.getVehicleReport(params.id, session.user.id, { filter, from, to });
    return NextResponse.json(report);
  } catch (error) {
    return handleApiError(error);
  }
}
```

### reportService.getVehicleReport()

[Source: architecture.md#6-components]

Add to `src/services/reportService.ts` (create file):

```typescript
async getVehicleReport(vehicleId: string, userId: string, filter: ReportFilter): Promise<VehicleReport> {
  // 1. Ownership check: throws ForbiddenError if user doesn't own or have access
  const vehicle = await vehicleService.getWithAccessCheck(vehicleId, userId);

  // 2. Build date range from filter
  const { from, to, prevFrom, prevTo } = resolveDateRange(filter);

  // 3. Fetch expenses in range
  const expenses = await expenseService.listByVehicleAndRange(vehicleId, from, to);
  const prevExpenses = await expenseService.listByVehicleAndRange(vehicleId, prevFrom, prevTo);

  // 4. Aggregate by category
  const byCategory = groupBy(expenses, e => e.category);
  const totalSpend = expenses.reduce((s, e) => s + convertToUserCurrency(e), 0);
  const prevTotal = prevExpenses.reduce((s, e) => s + convertToUserCurrency(e), 0);

  // 5. Monthly series for line chart
  const monthlyData = buildMonthlySeries(expenses, userId);

  // 6. Most expensive category
  const mostExpensive = Object.entries(byCategory)
    .sort(([,a],[,b]) => sumAmounts(b) - sumAmounts(a))[0]?.[0] ?? null;

  return {
    vehicle: { id: vehicle.id, name: vehicle.name },
    totalSpend,
    prevTotalSpend: prevTotal,
    avgMonthlySpend: calculateAvgMonthly(monthlyData),
    mostExpensiveCategory: mostExpensive,
    byCategory: formatCategoryData(byCategory),
    monthlyData,
    hadCurrencyConversion: expenses.some(e => e.currency !== userId_currency),
  };
}
```

All aggregation happens server-side; no raw amounts returned to client unless needed.

### Recharts Dynamic Import (Bundle Size Constraint)

[Source: architecture.md#8-performance-strategy]

Recharts MUST be loaded via `next/dynamic` — it is a large library and must NOT be statically imported at the top of any file:

```typescript
// src/components/reports/SpendChart.tsx
"use client";
import dynamic from "next/dynamic";
import { Skeleton } from "@/components/ui/skeleton";

const BarChartComponent = dynamic(
  () => import("recharts").then(m => ({ default: m.BarChart })),
  { ssr: false, loading: () => <Skeleton className="h-64 w-full" /> }
);
// Same pattern for LineChart, PieChart (Donut)
```

The `SpendChart` component receives `chartType: "bar" | "line" | "donut"` and `data` as props and renders the correct Recharts chart.

### Date Filter Logic

[Source: architecture.md#5-api-specification]

`resolveDateRange(filter)` in `src/services/reportService.ts`:

| `filter` value | Current range | Comparison range |
|---|---|---|
| `all` | inception → now | N/A (prevTotal = 0) |
| `year` | Jan 1 this year → today | Jan 1 last year → Dec 31 last year |
| `month` | 1st of current month → today | 1st of previous month → last day of previous month |
| `custom` | `from`→`to` from query params | same duration, shifted back |
