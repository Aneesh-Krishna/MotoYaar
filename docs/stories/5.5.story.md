## Dev Notes

### R2 Key Strategy for Receipts

[Source: architecture.md#7-external-apis (Cloudflare R2)]

Receipts differ from documents in that they are always kept (not parse-only). The R2 key pattern is:

```
{userId}/receipts/{expenseId}/{uuid}.{ext}
```

For new expenses, the `expenseId` is only available after `POST /api/vehicles/[id]/expenses` returns the created record. The upload flow is therefore:

1. **Add Expense flow:** User selects a receipt file. Client requests a pre-signed URL with a temp path (`{userId}/receipts/temp/{uuid}.{ext}`). File is uploaded directly to R2. The temp key is stored in form state.
2. On expense save: the temp key is passed in the POST body. The server moves/renames the file to the permanent key `{userId}/receipts/{expenseId}/{filename}` after creating the expense record.

For **Edit Expense:** the `expenseId` is already known, so the permanent key can be used directly.

### POST /api/uploads/receipt

[Source: architecture.md#5-api-specification]

Create `src/app/api/uploads/receipt/route.ts`:

```typescript
export async function POST(req: Request) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    const { filename, contentType } = await req.json();

    // Validate content type
    const allowed = ["image/jpeg", "image/png", "application/pdf"];
    if (!allowed.includes(contentType)) {
      return NextResponse.json({ error: "Invalid file type" }, { status: 400 });
    }

    const ext = contentType === "application/pdf" ? "pdf"
              : contentType === "image/png" ? "png" : "jpg";
    const tempKey = `${session.user.id}/receipts/temp/${crypto.randomUUID()}.${ext}`;

    const uploadUrl = await generateUploadUrl(tempKey, contentType, 5 * 60); // 5-min TTL

    return NextResponse.json({ uploadUrl, tempKey });
  } catch (error) {
    return handleApiError(error);
  }
}
```

`generateUploadUrl()` in `src/lib/r2.ts` (implement alongside `generateAccessUrl()`):

```typescript
import { PutObjectCommand } from "@aws-sdk/client-s3";
import { getSignedUrl } from "@aws-sdk/s3-request-presigner";

export async function generateUploadUrl(
  key: string,
  contentType: string,
  expiresInSeconds = 300
): Promise<string> {
  const command = new PutObjectCommand({
    Bucket: process.env.R2_BUCKET_NAME!,
    Key: key,
    ContentType: contentType,
  });
  return getSignedUrl(s3Client, command, { expiresIn: expiresInSeconds });
}
```

### Expense Save with Receipt — Add Flow

[Source: architecture.md#8-core-workflows]

Update `expenseService.create()` (Story 5.1) to handle the temp receipt key:

```typescript
async create(userId: string, vehicleId: string | undefined, data: CreateExpenseInput): Promise<Expense> {
  const [expense] = await db.insert(expenses).values({
    userId,
    vehicleId: vehicleId ?? null,
    price: data.price,
    currency: "INR",
    date: new Date(data.date),
    reason: data.reason,
    whereText: data.whereText ?? null,
    comment: data.comment ?? null,
    receiptUrl: null,
    receiptKey: null,
  }).returning();

  // Move receipt from temp path to permanent path if provided
  if (data.tempReceiptKey) {
    const ext = data.tempReceiptKey.split(".").pop() ?? "jpg";
    const permanentKey = `${userId}/receipts/${expense.id}/${crypto.randomUUID()}.${ext}`;
    await storageService.copyFile(data.tempReceiptKey, permanentKey);
    await storageService.deleteFile(data.tempReceiptKey).catch((e) =>
      logger.error({ e }, "Failed to delete temp receipt")
    );

    // Update expense with receipt info
    const [updated] = await db.update(expenses)
      .set({ receiptKey: permanentKey, receiptUrl: permanentKey }) // receiptUrl = key for now; signed URL generated on demand
      .where(eq(expenses.id, expense.id))
      .returning();
    return updated;
  }

  return expense;
}
```

```typescript
// In update():
if (data.tempReceiptKey) {
  // Delete old receipt if exists
  if (expense.receiptKey) {
    await storageService.deleteFile(expense.receiptKey).catch((e) =>
      logger.error({ e }, "Failed to delete old receipt")
    );
  }
  // Move new temp file to permanent
  const ext = data.tempReceiptKey.split(".").pop() ?? "jpg";
  const permanentKey = `${userId}/receipts/${expenseId}/${crypto.randomUUID()}.${ext}`;
  await storageService.copyFile(data.tempReceiptKey, permanentKey);
  await storageService.deleteFile(data.tempReceiptKey).catch(logger.error);
  receiptKey = permanentKey;
}

if (data.removeReceipt && expense.receiptKey) {
  await storageService.deleteFile(expense.receiptKey).catch(logger.error);
  receiptKey = null;
}
```
