## Story

**As a** vehicle owner,
**I want** to log how many litres I filled and my odometer reading each time I fuel up,
**so that** MotoYaar automatically calculates and tracks my fuel efficiency (kmpl) over time.

---

## Acceptance Criteria

1. - [ ] When `reason = "Fuel"` is selected on the expense form, two additional optional fields appear: **Litres filled** (decimal number) and **Odometer reading** (integer km)
2. - [ ] Both fields are optional — existing fuel expense flow works unchanged if left blank
3. - [ ] When both litres and current odometer are provided, and a previous fuel expense with odometer exists for the same vehicle, the system calculates: `kmpl = (current_odometer - previous_odometer) / litres_filled`
4. - [ ] The calculated kmpl is stored on the expense record and shown on the fuel expense list item: *"23.4 kmpl"*
5. - [ ] The vehicle detail Overview tab shows **current average fuel efficiency** (mean of all calculated kmpl values for that vehicle, last 5 fill-ups)
6. - [ ] The vehicle detail Overview tab shows the **last fill-up date and litres**
7. - [ ] If only litres are provided (no odometer), litres are stored but kmpl is not calculated; no error shown
8. - [ ] If only odometer is provided (no litres), odometer is stored for future delta calculation; kmpl not calculated
9. - [ ] `pnpm typecheck` and `pnpm lint` pass

### kmpl Calculation Logic (in expenseService)

```typescript
async function calculateKmpl(
  userId: string,
  vehicleId: string,
  currentOdometer: number,
  litresFilled: number
): Promise<number | null> {
  // Find the most recent fuel expense for this vehicle with an odometer reading
  const prev = await db.query.expenses.findFirst({
    where: and(
      eq(expenses.vehicleId, vehicleId),
      eq(expenses.reason, 'Fuel'),
      isNotNull(expenses.odometerKm)
    ),
    orderBy: desc(expenses.date),
  });

  if (!prev?.odometerKm) return null;
  const distanceKm = currentOdometer - prev.odometerKm;
  if (distanceKm <= 0) return null; // odometer went backwards — ignore
  return parseFloat((distanceKm / litresFilled).toFixed(2));
}
```

### Vehicle Overview — Fuel Efficiency Display

Query: last 5 fuel expenses with calculated kmpl for the vehicle.
```typescript
const fuelEfficiency = await db
  .select({ kmpl: expenses.kmpl })
  .from(expenses)
  .where(and(
    eq(expenses.vehicleId, vehicleId),
    eq(expenses.reason, 'Fuel'),
    isNotNull(expenses.kmpl)
  ))
  .orderBy(desc(expenses.date))
  .limit(5);

const avgKmpl = fuelEfficiency.length
  ? fuelEfficiency.reduce((s, r) => s + r.kmpl!, 0) / fuelEfficiency.length
  : null;
```
