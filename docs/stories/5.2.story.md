# Story 5.2 — Expense List per Vehicle

**Epic:** 05 — Expense Tracking
**Story:** 5.2
**Status:** Draft
**Created:** 2026-03-15
**Depends On:** Story 5.1 (expenses can be created before listing is useful)

## Story

**As a** vehicle owner,
**I want** to see all expenses for a vehicle in one list,
**so that** I can review what I've spent on it over time.

---

### expenseService.listByVehicle()

[Source: architecture.md#6-components]

Add to `src/services/expenseService.ts`:

```typescript
async listByVehicle(vehicleId: string, userId: string): Promise<Expense[]> {
  // Access check (owner or viewer)
  await vehicleService.getWithAccessCheck(vehicleId, userId);

  return db.query.expenses.findMany({
    where: eq(expenses.vehicleId, vehicleId),
    orderBy: [desc(expenses.date)],
  });
}
```

```typescript
export async function GET(req: Request, { params }: { params: { id: string } }) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    const expenseList = await expenseService.listByVehicle(params.id, session.user.id);
    return NextResponse.json(expenseList);
  } catch (error) {
    return handleApiError(error);
  }
}
```

**Price formatting** (Indian number system — lakhs/crores):
```typescript
// src/utils/currency.ts
export function formatINR(amount: number): string {
  return new Intl.NumberFormat("en-IN", {
    style: "currency",
    currency: "INR",
    maximumFractionDigits: 0,
  }).format(amount);
}
// ₹1,23,456 format (Indian locale grouping)
```

**Reason badge** colors:
| Reason | Color |
|--------|-------|
| Service | Blue — `bg-blue-50 text-blue-700` |
| Fuel | Yellow — `bg-yellow-50 text-yellow-700` |
| Trip | Orange — `bg-orange-50 text-orange-700` |
| Others | Gray — `bg-gray-100 text-gray-600` |

```tsx
<button
  onClick={() => onTap(expense.id)}
  className="w-full text-left px-4 py-3 flex items-center justify-between active:bg-gray-50"
>

    <div className="flex items-center gap-2">

      )}

    <span className="text-xs text-gray-400">{format(new Date(expense.date), "d MMM yyyy")}</span>
  </div>
```
