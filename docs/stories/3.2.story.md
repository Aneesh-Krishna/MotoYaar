## Dev Notes

### Dashboard Page Structure

[Source: architecture.md#12-unified-project-structure]

`src/app/(app)/page.tsx` already exists as a stub. Implement it as a React Server Component. The page should:
1. Get the session via `getServerSession(authOptions)` — redirect to `/login` if null
2. Fetch `vehicles` via `vehicleService.listByUser(userId)` — includes `totalSpend`, `nextDocumentExpiry`, `nextDocumentStatus` computed fields
3. Fetch recent activity: last 5 entries from `expenseService.recentByUser(userId, 5)` + `tripService.recentByUser(userId, 5)` merged and sorted by date
4. Pass data as props to client sub-components

```tsx
// src/app/(app)/page.tsx (RSC)
import { getServerSession } from "next-auth";
import { authOptions } from "@/lib/auth";
import { vehicleService } from "@/services/vehicleService";
import { expenseService } from "@/services/expenseService";
import { redirect } from "next/navigation";

export default async function DashboardPage() {
  const session = await getServerSession(authOptions);
  if (!session) redirect("/login");

  const [vehicles, recentExpenses] = await Promise.all([
    vehicleService.listByUser(session.user.id),
    expenseService.recentByUser(session.user.id, 5),
  ]);

  return (
    <div className="px-4 py-4 space-y-6">
      <GreetingSection userName={session.user.name} />
      <VehicleCarousel vehicles={vehicles} />
      <AlertsStrip vehicles={vehicles} />
      <RecentActivity expenses={recentExpenses} />
      <CommunityHighlights />
    </div>
  );
}
```

| Status | Color | Icon (Lucide) | Label | Tailwind classes |
|--------|-------|---------------|-------|-----------------|

### Alerts Strip

[Source: front-end-spec.md#4-core-screen-specifications]

Show a horizontal strip below the vehicle carousel if any vehicle has `nextDocumentStatus === 'expiring'` or `'expired'`:

```tsx
// Amber banner for expiring; red for expired
<div className={cn("rounded-lg px-4 py-3 flex items-center gap-2 text-sm font-medium",
  hasExpired ? "bg-red-50 text-red-700" : "bg-amber-50 text-amber-700"
)}>
  <AlertTriangle size={16} />
  {hasExpired
    ? `${expiredCount} document(s) have expired. Renew now.`
    : `${expiringCount} document(s) expiring soon. Check your vehicles.`
  }
</div>
```

If no documents are expiring or expired: render nothing.

### Pull-to-Refresh

[Source: front-end-spec.md#6-responsive-design--pwa-considerations]

Wrap the dashboard content in a client component `DashboardRefreshWrapper` that uses the `router.refresh()` pattern from Next.js App Router for pull-to-refresh on mobile. Use a simple swipe-down gesture detection or a library like `react-pull-to-refresh`. This is a progressive enhancement — if not immediately feasible, implement with a "Refresh" button at the top of the page as a fallback, and note the pull-gesture as a follow-up.

```tsx
"use client";
import { useRouter } from "next/navigation";

export function RefreshButton() {
  const router = useRouter();
  return (
    <button onClick={() => router.refresh()} className="text-xs text-gray-400 underline">
      Refresh
    </button>
  );
}
```
