## Story

**As a** user,
**I want** to log a trip with all relevant details,
**so that** I have a complete record of my journeys and their cost.

---

## Acceptance Criteria

1. - [ ] Trip form accessible from: Trips tab → FAB; Vehicle Detail → Trips tab → "Add Trip" CTA; and when "Trip" chip is selected in the expense form (redirected from Story 5.1)
2. - [ ] Trip form opens in a BottomSheet on mobile (same BottomSheet component)
3. - [ ] Fields: Title (required), Description (optional), Date (required — toggle between single day and date range), Vehicle (optional dropdown of user's vehicles; pre-filled if opened from vehicle detail), Route (optional free-text field + optional maps link field), Time taken (optional, HH:MM format input), Expense breakdown (optional, repeatable rows: category dropdown + amount)
4. - [ ] Expense breakdown categories: Food / Fuel / Stay / Toll / Other
5. - [ ] On save WITH vehicle linked AND breakdown has amounts: one combined expense auto-created under that vehicle (sum of all breakdown amounts; `reason = "Trip"`, `tripId = trip.id`)
6. - [ ] On save WITH NO vehicle linked AND breakdown has amounts: prompt shown "This trip has no vehicle linked. Add it as a general expense?" — Yes → create expense with `vehicleId = null`; No → skip expense creation
7. - [ ] On save WITHOUT any breakdown amounts: no expense created; no prompt shown
8. - [ ] `POST /api/trips` route; `tripService.create(userId, data)` handles expense auto-creation logic
9. - [ ] Zod schema: `createTripSchema` in `src/lib/validations/trip.ts`
10. - [ ] On successful save: redirect to `/trips/[id]` (trip detail page)
11. - [ ] `pnpm typecheck` and `pnpm lint` pass

### tripService.create()

[Source: architecture.md#6-components]

Add to `src/services/tripService.ts`:

```typescript
async create(userId: string, data: CreateTripInput): Promise<Trip> {
  const totalCost = data.breakdown.reduce((sum, item) => sum + item.amount, 0);

  // 1. Insert trip record
  const [trip] = await db.insert(trips).values({
    userId,
    vehicleId: data.vehicleId ?? null,
    title: data.title,
    description: data.description ?? null,
    startDate: new Date(data.startDate),
    endDate: data.endDate ? new Date(data.endDate) : null,
    routeText: data.routeText ?? null,
    mapsLink: data.mapsLink ?? null,
    timeTaken: data.timeTaken ?? null,
    breakdown: data.breakdown,
  }).returning();

  // 2. Auto-create combined expense if breakdown has amounts
  if (totalCost > 0) {
    if (data.vehicleId) {
      // Vehicle linked → expense under vehicle
      await expenseService.create(userId, data.vehicleId, {
        price: totalCost,
        date: data.startDate,
        reason: "Trip",
        whereText: data.title,
        tripId: trip.id,    // links expense to trip
      });
    }
    // Note: if no vehicleId but amounts exist, the frontend handles the prompt (AC 6)
    // The API receives either: no vehicleId (skip expense), or vehicleId=null + createGeneralExpense=true
    if (!data.vehicleId && data.createGeneralExpense) {
      await expenseService.create(userId, undefined, {
        price: totalCost,
        date: data.startDate,
        reason: "Trip",
        whereText: data.title,
        tripId: trip.id,
      });
    }
  }

  return trip;
}
```

The `tripId` column on expenses links them to their parent trip. Add `tripId` to `createExpenseSchema` as an optional field used only by `tripService`.

const handleSave = async (data: CreateTripInput) => {

### Zod Schema

[Source: architecture.md#17-coding-standards]

Create `src/lib/validations/trip.ts`:

```typescript
import { z } from "zod";

const tripBreakdownItemSchema = z.object({
  category: z.enum(["Food", "Fuel", "Stay", "Toll", "Other"]),
  amount: z.number().min(0),
});

export const createTripSchema = z.object({
  title: z.string().min(1, "Title is required"),
  description: z.string().optional(),
  startDate: z.string().min(1, "Date is required"),
  endDate: z.string().optional(),
  vehicleId: z.string().uuid().optional(),
  routeText: z.string().optional(),
  mapsLink: z.string().url().optional().or(z.literal("")),
  timeTaken: z.string().optional(),              // "02:30" format
  breakdown: z.array(tripBreakdownItemSchema).default([]),
  createGeneralExpense: z.boolean().optional(),  // from no-vehicle prompt
});

export const updateTripSchema = createTripSchema.partial();

export type CreateTripInput = z.infer<typeof createTripSchema>;
export type UpdateTripInput = z.infer<typeof updateTripSchema>;
```
