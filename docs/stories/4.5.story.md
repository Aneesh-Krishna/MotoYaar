# Story 4.5 — Edit and Delete Document

**Epic:** 04 — Document Management
**Story:** 4.5
**Status:** Draft
**Created:** 2026-03-15
**Depends On:** Story 4.3 (Document List — edit/delete initiated from the list)

---

## Story

**As a** vehicle owner,
**I want** to edit or delete a document I've uploaded,
**so that** I can correct wrong expiry dates or remove outdated documents.

---

## Acceptance Criteria

1. - [ ] Edit: tapping a document row opens a form pre-filled with document type and expiry date (in a BottomSheet)
2. - [ ] Edit form allows: changing document type, changing expiry date (date picker), re-uploading a new file (triggers AI parse again using the Story 4.1 flow)
3. - [ ] On save edit: `PATCH /api/documents/[id]` called; `documentService.update()` handles DB update
4. - [ ] Delete: accessible from the document row kebab menu ("Delete"); `ConfirmModal` shown: "Remove this document?"
5. - [ ] On delete confirm: `DELETE /api/documents/[id]` called; `documentService.delete()` handles DB delete + R2 file deletion if `storageKey` exists
6. - [ ] Deleting a document has no cascade effects — only this document record is deleted
7. - [ ] After edit: document list refreshes to show updated data
8. - [ ] After delete: document removed from list; if it was the last document, empty state shown
9. - [ ] `pnpm typecheck` and `pnpm lint` pass

### PATCH /api/documents/[id]

[Source: architecture.md#5-api-specification]

```typescript
export async function PATCH(req: Request, { params }: { params: { id: string } }) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    const body = await req.json();
    const data = updateDocumentSchema.parse(body);
    const doc = await documentService.update(params.id, session.user.id, data);
    return NextResponse.json(doc);
  } catch (error) {
    return handleApiError(error);
  }
}

export async function DELETE(req: Request, { params }: { params: { id: string } }) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    await documentService.delete(params.id, session.user.id);
    return new NextResponse(null, { status: 204 });
  } catch (error) {
    return handleApiError(error);
  }
}
```

### documentService.update()

[Source: architecture.md#6-components]

Add to `src/services/documentService.ts`:

```typescript
async update(docId: string, userId: string, data: UpdateDocumentInput): Promise<Document> {
  // Ownership check via vehicle
  const doc = await db.query.documents.findFirst({ where: eq(documents.id, docId) });
  if (!doc) throw new NotFoundError("Document not found");

  // Check access (owner or viewer — but only owner should edit)
  const vehicle = doc.vehicleId
    ? await db.query.vehicles.findFirst({ where: eq(vehicles.id, doc.vehicleId) })
    : null;

  if (vehicle && vehicle.userId !== userId) {
    // Check if it's a DL (vehicleId = null) — belongs to user directly
    throw new ForbiddenError("Only the vehicle owner can edit this document");
  }
  if (!vehicle && doc.userId !== userId) {
    throw new ForbiddenError("Only the document owner can edit it");
  }

  // Recompute status if expiryDate changed
  const user = await db.query.users.findFirst({ where: eq(users.id, userId) });
  const newStatus = computeDocumentStatus(
    data.expiryDate ?? null,
    user?.notificationWindowDays ?? 30
  );

  const [updated] = await db.update(documents)
    .set({
      type: data.type,
      label: data.label,
      expiryDate: data.expiryDate ? new Date(data.expiryDate) : null,
      parseStatus: data.expiryDate ? "manual" : "incomplete",
      status: newStatus,
    })
    .where(eq(documents.id, docId))
    .returning();

  return updated;
}
```

### documentService.delete()

[Source: architecture.md#6-components]

Add to `src/services/documentService.ts`:

```typescript
async delete(docId: string, userId: string): Promise<void> {
  const doc = await db.query.documents.findFirst({ where: eq(documents.id, docId) });
  if (!doc) throw new NotFoundError("Document not found");

  // Ownership check
  if (doc.vehicleId) {
    const vehicle = await db.query.vehicles.findFirst({ where: eq(vehicles.id, doc.vehicleId) });
    if (!vehicle || vehicle.userId !== userId) {
      throw new ForbiddenError("Only the vehicle owner can delete this document");
    }
  } else {
    // DL or user-level document
    if (doc.userId !== userId) throw new ForbiddenError("Only the document owner can delete it");
  }

  // Delete R2 file if stored
  if (doc.storageKey) {
    await storageService.deleteFile(doc.storageKey).catch((e) =>
      logger.error({ error: e, docId }, "Failed to delete document R2 file")
    );
  }

  // Delete DB record
  await db.delete(documents).where(eq(documents.id, docId));
}
```

No cascade effects — only this document record is deleted. The parent vehicle is unaffected.

### Zod Schema for Update

[Source: architecture.md#17-coding-standards]

Add to `src/lib/validations/document.ts`:

```typescript
export const updateDocumentSchema = z.object({
  type: z.enum(["RC", "Insurance", "PUC", "DL", "Other"]).optional(),
  label: z.string().optional(),
  expiryDate: z.string().optional().nullable(),  // null = remove expiry date
  tempR2Key: z.string().optional(),              // if file was replaced
});

export type UpdateDocumentInput = z.infer<typeof updateDocumentSchema>;
```
