## Story

**As a** user who triggered an AI report,
**I want** to read the generated narrative analysis,
**so that** I can review AI-powered insights about my spending habits.

1. - [ ] Route `src/app/(app)/reports/ai/[id]/page.tsx` exists (create new); Server Component

10. - [ ] History list accessible from Reports page sidebar/header link

### Report Detail Page

[Source: architecture.md#12-unified-project-structure]

Create `src/app/(app)/reports/ai/[id]/page.tsx` as a Server Component:

```tsx
// src/app/(app)/reports/ai/[id]/page.tsx (RSC)
import { getServerSession } from "next-auth";
import { authOptions } from "@/lib/auth";
import { aiReportService } from "@/services/aiReportService";
import { redirect, notFound } from "next/navigation";

interface PageProps {
  params: { id: string };
}

export default async function AIReportDetailPage({ params }: PageProps) {
  const session = await getServerSession(authOptions);
  if (!session) redirect("/login");

  let report;
  try {
    report = await aiReportService.getById(params.id, session.user.id);
  } catch (e) {
    notFound();
  }

  return <AIReportDetailView report={report} />;
}
```

`AIReportDetailView` is a `"use client"` component only if interactivity is needed; otherwise keep it as a server-side render with static HTML.

### aiReportService Methods

[Source: architecture.md#6-components]

Add to `src/services/aiReportService.ts`:

```typescript
async getById(reportId: string, userId: string): Promise<AiReport> {
  const report = await db.query.aiReports.findFirst({
    where: eq(aiReports.id, reportId),
  });
  if (!report) throw new NotFoundError("AI report not found");
  if (report.userId !== userId) throw new ForbiddenError("Access denied");
  return report;
}

async listByUser(userId: string): Promise<AiReport[]> {
  return db.query.aiReports.findMany({
    where: eq(aiReports.userId, userId),
    orderBy: [desc(aiReports.requestedAt)],
  });
}
```
