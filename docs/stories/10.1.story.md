## Story

**As a** vehicle owner,
**I want** to invite another person to view my vehicle's details,
**so that** I can share maintenance history, documents, and trip records with a family member, mechanic, or co-owner.

  ON vehicle_invites(vehicle_id, invitee_email)

### vehicleInviteService.createInvite()

[Source: architecture.md#6-components]

Create `src/services/vehicleInviteService.ts`:

```typescript
async createInvite(
  ownerUserId: string,
  vehicleId: string,
  inviteeEmail: string
): Promise<VehicleInvite> {
  // 1. Ownership check
  const vehicle = await vehicleService.getWithAccessCheck(vehicleId, ownerUserId);

  // 2. Cannot invite self
  const owner = await userService.getById(ownerUserId);
  if (owner.email.toLowerCase() === inviteeEmail.toLowerCase()) {
    throw new ConflictError("You cannot invite yourself");
  }

  // 3. Duplicate pending check
  const existing = await db.query.vehicleInvites.findFirst({
    where: and(
      eq(vehicleInvites.vehicleId, vehicleId),
      eq(vehicleInvites.inviteeEmail, inviteeEmail.toLowerCase()),
      eq(vehicleInvites.status, "pending"),
    ),
  });
  if (existing) throw new ConflictError("A pending invite already exists for this email");

  // 4. Create invite
  const expiresAt = new Date(Date.now() + 3 * 24 * 60 * 60 * 1000); // +3 days
  const [invite] = await db.insert(vehicleInvites).values({
    vehicleId,
    ownerUserId,
    inviteeEmail: inviteeEmail.toLowerCase(),
    expiresAt,
  }).returning();

  // 5. Send email
  const inviteUrl = `https://motoyaar.app/invites/${invite.token}`;
  await emailService.sendVehicleInviteEmail(inviteeEmail, vehicle.name, inviteUrl);

  return invite;
}
```

### POST /api/invites

[Source: architecture.md#5-api-specification]

Create `src/app/api/invites/route.ts`:

```typescript
import { z } from "zod";

const createInviteSchema = z.object({
  vehicleId: z.string().uuid(),
  inviteeEmail: z.string().email(),
});

export async function POST(req: Request) {
  const session = await getServerSession(authOptions);
  if (!session) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

  try {
    const body = await req.json();
    const { vehicleId, inviteeEmail } = createInviteSchema.parse(body);
    const invite = await vehicleInviteService.createInvite(session.user.id, vehicleId, inviteeEmail);
    return NextResponse.json(invite, { status: 201 });
  } catch (error) {
    return handleApiError(error);
  }
}
```

### Manage Sharing Page

[Source: architecture.md#12-unified-project-structure]

Create `src/app/(app)/garage/[id]/sharing/page.tsx` as a Server Component:

```tsx
export default async function SharingPage({ params }: { params: { id: string } }) {
  const session = await getServerSession(authOptions);
  if (!session) redirect("/login");

  const [vehicle, access, pendingInvites] = await Promise.all([
    vehicleService.getWithAccessCheck(params.id, session.user.id),
    vehicleInviteService.listAccess(params.id, session.user.id),
    vehicleInviteService.listPendingInvites(params.id, session.user.id),
  ]);

  return (
    <SharingPageView
      vehicle={vehicle}
      access={access}
      pendingInvites={pendingInvites}
    />
  );
}
```

`SharingPageView` is a `"use client"` component with the invite email input and list of current access entries.

`listAccess` and `listPendingInvites` are added to `vehicleInviteService` (also used in Story 10.4).
